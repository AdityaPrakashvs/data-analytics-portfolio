# 📊 UPI Transactions Data Analysis

## 📌 Objective
To analyze UPI transaction data and uncover trends in user behavior, transaction volume, and payment patterns.

## 🛠 Tools Used
- Power BI  
- Excel  

## 📂 Dataset
- UPI transactions dataset including transaction amount, date, users, and categories.

## 🔍 Approach
- Cleaned and transformed data using Excel
- Analyzed transaction patterns over time
- Built interactive dashboard in Power BI
- Identified high-frequency usage patterns

## 📊 Key Insights
- Transaction volume shows growth over time
- Peak transactions occur during specific periods
- Certain categories dominate UPI usage
- High-frequency users contribute significantly to total volume
